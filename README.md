AI-Based Waste Segregation Awareness and Advisory System
Overview

This project is an AI-powered system that helps users learn proper waste segregation, classifies waste items, and provides correct disposal guidance. It promotes recycling, environmental awareness, and sustainable waste management practices, aligning with global sustainability goals.

Purpose

To educate communities, reduce pollution, encourage responsible consumption, and demonstrate the use of AI for environmental sustainability.

Impact

Improves waste segregation habits

Reduces landfill waste

Encourages eco-friendly behavior

Supports SDG 11, 12, and 13

Future Scope

Image-based waste recognition

Mobile app and voice-based AI
